Accessibility · WCAG 2.1 AA

Approved contrast pairs

Ink on white15.8:1 ✓ AAA
Muted on white6.6:1 ✓ AA
Teal-800 link on white5.6:1 ✓ AA
White on navy11.4:1 ✓ AAA
Navy-900 on cyan8.1:1 ✓ AAA

Rules baked in

✓Focus visible — 3px cyan-300 ring, 2px offset, on every interactive element.
✓Touch targets — minimum 44×44px on mobile.
✓Semantic HTML — one h1/page, ordered h2–h6, landmarks (header/nav/main/footer).
✓Forms — every input has a <label>; errors use aria-invalid + aria-describedby.
✓Reduced motion — animations honour prefers-reduced-motion.
✓RTL — logical properties (margin-inline, padding-inline) so Arabic mirrors cleanly.
✓Images — descriptive alt text; decorative art marked aria-hidden.
Focus ring spec outline: 3px solid cyan-300; offset: 2px
06

Tailwind v4 tokens

Drop this into your app.css. Tailwind v4 reads @theme and generates the utility classes used across the Blade templates.

@import "tailwindcss";

@theme {
  /* Brand */
  --color-navy-50:  #eff5f9;   --color-navy-500: #356a95;
  --color-navy-100: #dde9f2;   --color-navy-600: #27567f;
  --color-navy-200: #b8d0e2;   --color-navy-700: #1c4068;  /* brand */
  --color-navy-300: #84aac8;   --color-navy-800: #143150;
  --color-navy-400: #5286ad;   --color-navy-900: #0e2236;

  --color-cyan-100: #d3f4f5;   --color-cyan-500: #1cb5bb;
  --color-cyan-200: #a6e9eb;   --color-cyan-600: #11a0a6;
  --color-cyan-300: #6fdadd;   --color-cyan-700: #0b6f74;  /* a11y text */
  --color-cyan-400: #34c8cd;   --color-cyan-800: #0c5f63;  /* brand */

  --color-ink:    #0f2230;     --color-muted:  #51616c;
  --color-line:   #e3edf1;     --color-surface:#f6fafb;

  --color-success:#1f8a5b;  --color-warning:#b7791f;  --color-error:#c4332b;

  /* Type */
  --font-sans: "Plus Jakarta Sans", system-ui, sans-serif;
  --font-arabic: "IBM Plex Sans Arabic", sans-serif;

  /* Radius */
  --radius-md: 12px;  --radius-lg: 16px;  --radius-xl: 22px;  --radius-2xl: 28px;

  /* Elevation */
  --shadow-md: 0 8px 24px -12px rgb(15 34 48 / .18);
  --shadow-lg: 0 22px 46px -26px rgb(15 34 48 / .30);
  --shadow-glow: 0 16px 34px -14px rgb(52 200 205 / .55);

  /* Motion */
  --ease-emphasized: cubic-bezier(.16, 1, .3, 1);
}

/* Respect reduced motion globally */
@media (prefers-reduced-motion: reduce) {
  *, ::before, ::after { animation-duration: .01ms !important; transition-duration: .01ms !important; }
}
